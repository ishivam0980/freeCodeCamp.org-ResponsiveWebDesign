Introduction

Python is a high-level, interpreted programming language with dynamic semantics. Its high-level built-in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development.

Python's simple, easy-to-learn syntax emphasizes readability and therefore reduces the cost of program maintenance. Python supports modules and packages, which encourages program modularity and code reuse.

The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.

print("Hello, World!")
  • Easy to learn and use
  • Extensive standard library
  • Cross-platform compatibility
Getting Started

To get started with Python, you need to install the Python interpreter on your system. Python is available for Windows, macOS, and Linux operating systems.

You can download Python from the official website at python.org. The installation process is straightforward and includes the Python interpreter, standard library, and IDLE (Integrated Development and Learning Environment).

Once installed, you can run Python programs from the command line or use an integrated development environment (IDE) like PyCharm, Visual Studio Code, or IDLE.

python --version

This command will display the installed Python version. You can also start the Python interactive interpreter by typing python in your terminal.

  • Download from python.org
  • Install the interpreter
  • Choose an IDE or text editor
  • Run your first program
Variables and Data Types

Python is dynamically typed, which means you don't need to declare variable types explicitly. The interpreter determines the type based on the assigned value.

Python supports several built-in data types including integers, floats, strings, booleans, lists, tuples, dictionaries, and sets.

# Variable assignments name = "Alice" age = 25 height = 5.6 is_student = True

Variables in Python are created when you first assign a value to them. Python uses duck typing, which means "if it looks like a duck and quacks like a duck, it's a duck."

  • Integers (int): Whole numbers
  • Floats (float): Decimal numbers
  • Strings (str): Text data
  • Booleans (bool): True or False
  • Lists: Ordered, mutable collections
  • Tuples: Ordered, immutable collections
Control Structures

Control structures in Python allow you to control the flow of program execution. The main control structures are conditional statements and loops.

Python uses indentation to define code blocks, which makes the code more readable and enforces good programming practices.

# If statement if age >= 18: print("You are an adult") elif age >= 13: print("You are a teenager") else: print("You are a child")

Python supports for loops and while loops for iteration. For loops are commonly used to iterate over sequences like lists, tuples, or strings.

# For loop for i in range(5): print(f"Number: {i}") # While loop count = 0 while count < 5: print(count) count += 1
  • if, elif, else statements
  • for loops
  • while loops
  • break and continue statements
Functions

Functions in Python are defined using the def keyword. They allow you to organize code into reusable blocks and follow the DRY (Don't Repeat Yourself) principle.

Functions can accept parameters and return values. Python supports default parameters, variable-length arguments, and keyword arguments.

def greet(name, greeting="Hello"): return f"{greeting}, {name}!" # Function call message = greet("Alice") print(message) # Output: Hello, Alice!

Python functions are first-class objects, meaning they can be assigned to variables, passed as arguments to other functions, and returned from functions.

Lambda functions provide a way to create small, anonymous functions. They are useful for short operations that can be defined inline.

# Lambda function square = lambda x: x ** 2 print(square(5)) # Output: 25
  • Function definition with def keyword
  • Parameters and return values
  • Default parameters
  • Variable-length arguments (*args, **kwargs)
  • Lambda functions
Object Oriented Programming

Python supports object-oriented programming (OOP) with classes and objects. OOP allows you to model real-world entities and create more organized and maintainable code.

A class is a blueprint for creating objects, while an object is an instance of a class. Classes can have attributes (data) and methods (functions).

class Person: def __init__(self, name, age): self.name = name self.age = age def introduce(self): return f"Hi, I'm {self.name} and I'm {self.age} years old." # Creating an object person = Person("Alice", 25) print(person.introduce())

Python supports inheritance, which allows you to create new classes based on existing ones. This promotes code reuse and creates hierarchical relationships between classes.

Encapsulation in Python is achieved through naming conventions. Attributes and methods starting with an underscore are considered private.

  • Constructors (__init__ method)
  • Instance and class attributes
